📊 Función get_quick_stats

Esta función calcula las estadísticas que se muestran en la página principal de eventos.

Si no existe, las estadísticas mostrarán 0 en lugar de los valores reales.

✅ Aplicar Función

Copia y pega el siguiente SQL en el SQL Editor de Supabase:

  1. Ve a tu Supabase Dashboard
  2. Selecciona tu proyecto
  3. Ve a la sección SQL Editor
  4. Crea una nueva query
  5. Copia y pega el SQL de abajo
  6. Haz clic en Run
-- RPC: get_quick_stats
create or replace function public.get_quick_stats()
returns jsonb language sql security definer as $$
with active_events as (
  select id, reward_points, end_at
  from public.events
  where status = 'active' and now() between start_at and end_at
), participants as (
  select count(distinct ep.user_id) as c
  from public.event_participants ep
  join active_events e on e.id = ep.event_id
)
select jsonb_build_object(
  'active_events_count', (select count(*) from active_events),
  'active_participants_count', (select c from participants),
  'points_in_play', coalesce((select sum(reward_points) from active_events), 0),
  'min_days_remaining', coalesce((select ceil(extract(epoch from min(end_at - now()))/86400.0)::int from active_events), 0)
);
$$;

comment on function public.get_quick_stats is 'Returns quick dashboard stats for the community events page.';

🧪 Verificar

Después de aplicar, verifica que funcione:

🎯 Qué Hace Esta Función

  • active_events_count: Número de eventos activos actualmente
  • active_participants_count: Total de participantes en eventos activos
  • points_in_play: Suma total de puntos de recompensa disponibles
  • min_days_remaining: Días restantes hasta el evento que termina primero
