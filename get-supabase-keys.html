Get Supabase API Keys

Follow these steps to get your Supabase API keys:

Step 1: Log in to your Supabase Dashboard

Go to https://app.supabase.com and log in to your account.

Step 2: Select your project

Click on your project "Potter Friends" in the dashboard.

Step 3: Go to API Settings

In the left sidebar, click on "Settings" and then "API".

Step 4: Copy your API Keys

You'll find two types of API keys:

  1. anon public - This is the key you should use in your frontend code
  2. service_role secret - This is for server-side operations only

For our authentication system, we need the anon public key.

Step 5: Update your code

Copy the anon public key and update the following files:

  • js/auth.js - Replace the SUPABASE_ANON_KEY value
  • supabase-mcp-server.js - Replace the supabaseKey value
  • .kiro/settings/mcp.json - Replace the SUPABASE_TOKEN value

Step 6: Create the necessary tables

Run the setup script to create the necessary tables:

node setup-supabase.js

Step 7: Test the connection

Run the test script to verify the connection:

node test-supabase-connection.js
